# PRODIGY_DS_05

## Traffic Accident Pattern Analysis

This project analyzes traffic accident data to identify patterns related to road conditions, weather, location, and time of day.

## Objective

- Analyze accident severity
- Identify accident hotspots
- Study weather conditions during accidents
- Analyze hourly accident trends
- Visualize accident patterns using Python

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Visualizations

- Accident Severity Distribution
- Accidents by Hour of Day
- Top Weather Conditions
- Top States with Most Accidents
- Top Cities with Most Accidents
- Correlation Heatmap
- Accident Severity under Different Weather Conditions

## Dataset

US Accidents Dataset (Kaggle)
